PM／井上 陽平（プラットヨネザワ） 申請主体／米沢市 R8第3回応募／6月中旬 更新日／2026-05-08
🔰 はじめての方へ — この事業の 5W1H

観光庁メニュー集 R8 から抽出した「○候補」3 補助金のうちの 1 つ。 国交省国土政策局所管の調査費（FS 補助）で、米沢駅前広場・道の駅米沢・上杉城史苑等の拠点整備事業化検討に 1/2 補助。 年 3 回募集なので 直近 R8 第 3 回（6 月中旬）が現実的ターゲット。

📌 What（何を取りに行く？）
国交省国土政策局所管の調査費補助（補助率 1/2）。地方公共団体が民間と連携する基盤整備事業の事業化検討（FS）に使える。R8 予算 398 百万円・年 3 回募集。
👥 Who（誰が動く？）
米沢市が申請主体（総合政策課・地方政策担当）／プラットヨネザワが伴走支援／民間連携事業者（JR 東日本仙台支社・上杉コーポレーション・道の駅米沢指定管理者・山形交通・米沢タクシー協会）の参画意向書が必須。
📅 When（いつ？）
★直近：R8 第 3 回応募（6/15 想定締切）→ 8 月上旬交付決定 → R8.8〜R9.3 調査実施 → R9.3 完了報告 → R9 以降 PPP/PFI 公募 → R10〜事業着手。
📍 Where（どこで？）
米沢駅前広場・道の駅米沢・上杉城史苑を結節点とした観光中核エリア。東町ポケットパーク・粡町ポケットパーク・シャトルバス停留所インフラも候補。
💡 Why（なぜやる？）
①レキマチハードは個別建造物に集中。本調査費で面的な拠点ネットワークの事業化検討を補完／②PPP/PFI 検討で民間活力導入の可能性を可視化／③360°オープンファクトリー本番（9/10-13）のリアル人流データを基礎データに組み込み可。
🛠 How（どう進める？）
①国交省国土政策局調整室への電話相談 → ②米沢市の申請主体合意 → ③連携事業者の参画意向書取得 → ④応募書類作成 → ⑤6/15 第 3 回応募 → ⑥交付決定 → ⑦コンサル選定 → ⑧概略設計＋PPP/PFI 検討。

事業の概要と狙い

官民連携基盤整備推進調査費＝国交省国土政策局所管の調査費で、地方公共団体が民間と連携して行う基盤整備事業の事業化検討（フィジビリティ調査）に必要な経費の1/2を補助。年3回募集・単年度執行。 対象事業は国交省所管の社会資本整備事業（道路・港湾・河川・公園・市街地整備等）。 支援対象の調査は、①概略設計、②基礎データ収集（地形・地質・交通量等）、③整備効果検討、④PPP/PFI導入可能性検討の4種。

米沢市の応募テーマ（仮）：「上杉の城下町 観光中核エリア整備事業」事業化検討。米沢駅前広場・道の駅米沢・上杉城史苑を結節点とした広域観光・交流拠点ネットワーク形成の概略設計＋PPP/PFI導入可能性検討。 R8重点配分項目②「広域的な観光又は交流拠点形成の促進に係る調査」に該当。
