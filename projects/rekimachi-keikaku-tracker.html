米沢市 歴史的風致維持向上計画 ＋ 街なみ環境整備事業

「上杉の城下町」を歴まち法に基づき面的整備／社総交（街なみ環境整備事業）の交付率上乗せ取得を狙う中期戦略
PM／井上 陽平（プラットヨネザワ） 計画策定主体／米沢市 認定目標／2028年（R10）前半 更新日／2026-05-08
🔰 はじめての方へ — この事業の 5W1H

観光庁メニュー集 R8 から抽出した「○候補」3 補助金のうちの 1 つ。 米沢市が「歴まち計画」を策定して 3 省同時認定を取得し、社総交（街なみ環境整備事業）等の交付率上乗せを獲得する中期戦略。 レキマチハード本体（既申請・R9.2 完了予定）の財源を二重化するための、裏方の準備プロジェクトです。

📌 What（何を取りに行く？）
市町村が策定し文部科学・農水・国交の 3 省が同時認定する法定計画（歴まち法／2008 年成立）。認定後、社総交（街なみ環境整備事業）／都市再生整備計画事業の交付率拡充（40→45%）等の上乗せ効果が発動。
👥 Who（誰が動く？）
米沢市が策定主体（都市計画課＋文化財担当課）／プラットヨネザワが伴走支援／3 省（国交省都市局・文化庁・農水省）が認定／有識者・上杉神社・伝国の杜・山形大・町内会が参画。
📅 When（いつ？）
R8.5（2026）打診・準備 → R8.9 米沢市 R9 予算化判断 → R8.12 推進協議会設置 → R9 計画素案・パブコメ → R10 前半（2028 春）3 省同時認定 → R10 秋 社総交予算反映。
📍 Where（どこで？）
上杉神社・米沢城跡（国史跡）・稽照殿（重文集積）を中核とする「上杉の城下町」の重点区域。粡町・東町・上杉神社周辺の面的整備にハマる。
💡 Why（なぜやる？）
①ハード本体（レキマチ）の整備財源を二重化／②社総交（街なみ整備）の歴史的風致形成建造物の修理補助対象化／③景観改善推進事業・歴史的観光資源高質化支援事業の発動／④米沢市の都市計画的な裏付け強化。
🛠 How（どう進める？）
①米沢市意向打診 → ②3 省事前相談 → ③推進協議会設置 → ④計画素案策定（7 章構成）→ ⑤パブコメ → ⑥3 省同時申請 → ⑦認定 → ⑧社総交予算反映。

事業の概要と狙い

歴まち計画＝市町村が策定し、文部科学大臣・農林水産大臣・国土交通大臣が共同認定する法定計画（歴史まちづくり法、2008年成立）。 米沢市が認定取得した場合の主な「上乗せ効果」は、 ①社総交（街なみ環境整備事業）の歴史的風致形成建造物の修理・買取が補助対象化（補助率1/2）、 ②社総交（都市再生整備計画事業）の交付率拡充（40%→45%）、 ③景観改善推進事業（既存不適格建築物の是正等）、 ④歴史的観光資源高質化支援事業（外観修景・空地除却・新築修景）。 レキマチハード本体（既申請、R9.2月完了予定）の「面的整備の財源を二重化する」戦略的位置付け。

♨ 南陽市版 — 赤湯熊野エリアの 5W1H

米沢市と並行して、南陽市の「熊野大社×赤湯温泉×赤湯駅」エリアでも歴まち計画策定を検討する案。 米沢市と異なり南陽市はDMO未認定／プラットが代表申請者・PMで伴走する構造になる見込み。 ベース資料は Notion『地域の観光資源充実のための環境整備推進事業（熊野大社）』および bunka-kanko-akayu（文化観光推進事業 南陽市地域計画）。

📌 What（何を取りに行く？）
米沢市と同じく歴まち法に基づく市町村計画の3省同時認定。認定後は社総交（街なみ環境整備事業）・歴史的観光資源高質化支援事業の対象化。R8観光庁レキマチ②整備メニュー（最大2億円）の活用要件としても歴まち計画認定が必要と既に北野宮司に説明済。
👥 Who（誰が動く？）
南陽市が策定主体（観光課・市長室）／熊野大社（北野宮司）が地域の旗振り役／プラットヨネザワが代表申請者・PM・事務局として伴走（米沢市と異なる立ち位置）／文化庁・農水省・国交省が3省共同認定／既に bunka-kanko-akayu で連携10事業者が同意済。
📅 When（いつ？）
R8.3/12 北野宮司との議題化 → R8.4/22 観光庁レキマチハード調査（熊野大社主体・1,000万円）提出済 → R8.5-6 南陽市役所打診 → R8.9 南陽市 R9 予算化判断 → R9 計画素案 → R10 前半 3省認定（米沢市と並列）。
📍 Where（どこで？）
熊野大社境内＋赤湯温泉街＋赤湯駅前＋烏帽子山公園＋吉野川沿い景観＋赤湯ワイナリーを含む南陽市全域。文化観光推進法地域計画（弊社既申請）の重点エリアと重なる。
💡 Why（なぜやる？）
①太々神楽（伊勢神宮と熊野大社のみに伝わる・ユネスコ無形文化遺産申請中）を歴史的風致の核として認定取得／②赤湯温泉街の街なみ整備の財源確保／③「上杉文化圏（米沢）×熊野信仰圏（南陽）＝置賜のルーツを巡る旅」の広域コンセプト実現／④R8観光庁レキマチハード②整備メニュー（最大2億円）の将来活用要件をクリア。
🛠 How（どう進める？）
①北野宮司から南陽市観光課・市長室への接続 → ②プラット＋熊野大社で南陽市役所訪問 → ③国交省＋文化庁＋農水省への3省事前相談 → ④南陽市の予算化判断 → ⑤推進体制構築 → ⑥計画素案策定 → ⑦3省同時申請 → ⑧認定。

  • 赤湯温泉街の街なみ整備ポテンシャル：歴まち認定後、社総交（街なみ環境整備事業）の歴史的風致形成建造物の修理対象化で、旅館の景観整備に補助率1/2が発動。
